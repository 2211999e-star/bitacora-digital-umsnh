## 🚀 CÓMO FUNCIONAN LAS MEJORAS

### 1. Importación de Archivos CSS

**style.css** ahora importa en orden:

```css
@import url("./design-system.css");          /* Base: tokens */
@import url("./premium-dashboard.css");      /* Componentes */
@import url("./premium-animations.css");     /* Animaciones */
@import url("./premium-enhancements.css");   /* Dashboard */
@import url("./premium-global.css");         /* Global */
```

### 2. Carga de JavaScript

En **index.html** se agregó:

```html
<script defer src="./js/premium-dashboard-components.js"></script>
```

Esto habilita:
- ✅ Animaciones de números en indicadores
- ✅ Actualización de hora en tiempo real
- ✅ Animaciones de scroll
- ✅ Ripple effects
- ✅ Tema light/dark persistente

### 3. Aplicación Gradual

Las mejoras se aplican **sobre la estructura HTML existente**, manteniendo total compatibilidad:

```
HTML Existente
        ↓
   CSS Premium
        ↓
   Interfaz Mejorada
```

No hay cambios en la funcionalidad, solo mejoras visuales.

## 💡 CÓMO USAR EL SISTEMA

### Para Crear Nuevas Tarjetas

```html
<!-- Tarjeta Grande (Importante) -->
<div class="card card-lg">
  <h3 class="title-md">Título</h3>
  <p class="body-base">Contenido</p>
</div>

<!-- Tarjeta Mediana (Normal) -->
<div class="card card-md">
  <!-- Contenido -->
</div>

<!-- Tarjeta Pequeña (Secundaria) -->
<div class="card card-sm">
  <!-- Contenido -->
</div>
```

### Para Crear Botones

```html
<button class="btn btn-primary">Acción Principal</button>
<button class="btn btn-secondary">Acción Secundaria</button>
<button class="btn btn-success">Éxito</button>
<button class="btn btn-danger">Peligro</button>
```

### Para Estados

```html
<!-- Operativo -->
<span class="badge badge-success">✓ Operativo</span>

<!-- Crítico -->
<span class="badge badge-danger">! Crítico</span>

<!-- Pendiente -->
<span class="badge badge-warning">⏱ Pendiente</span>
```

### Para Animaciones

```html
<!-- Entrada suave -->
<div class="animate-fade-in-up">Contenido</div>

<!-- Hover elegante -->
<div class="hover-lift">Hover para levantar</div>

<!-- Pulse continuo -->
<div class="animate-pulse">Pulsando</div>
```

## 🔧 PERSONALIZACIÓN

### Cambiar Colores Globales

Edita `design-system.css`:

```css
:root {
  --color-primary-600: #0284c7;    /* Cambiar primario */
  --color-success-600: #16a34a;    /* Cambiar success */
  --color-warning-600: #d97706;    /* Cambiar warning */
  /* ... */
}
```

### Cambiar Espaciado

```css
:root {
  --spacing-lg: 1.5rem;            /* Cambiar espaciado */
  --spacing-xl: 2rem;
  /* ... */
}
```

### Cambiar Tipografía

```css
:root {
  --font-size-base: 1rem;          /* Cambiar tamaño base */
  --font-size-2xl: 1.5rem;
  /* ... */
}
```
